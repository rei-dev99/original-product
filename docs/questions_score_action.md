# 診断質問（例）

## 分野
- ① AI活用習慣: 「エラーが出た時、まず何をしますか？ (A)公式ドキュメントを見る (B)AIにコードを全貼りする」といった行動特性クイズ
- ② アルゴリズム基礎: for文の回数計算、配列のインデックス操作、条件分岐の優先順位など
- ③ データベース: N+1問題の概念、外部キーの役割、JOINの基本
- ④ Web基礎: GET/POSTの違い、ステータスコード（404, 500等）、CookieとSessionの違い

## ① AI活用習慣

### Q1. エラーが出たとき、最初に何をしますか？
- A. エラー文を読み、原因を考えて検索する（3点）
- B. 自分でコードを見直す（2点）
- C. AIにそのまま貼り付ける（1点）

### Q2. AIの回答に対してどうしますか？
- A. 理解できるまで読み直す（3点）
- B. なんとなく理解して使う（2点）
- C. そのままコピペする（1点）

### Q3. AIの回答に対してどうしますか？
- A. 目的や前提を整理して質問する（3点）
- B. なんとなく聞く（2点）
- C. とりあえず丸投げ（1点）

## ② アルゴリズム基礎

### Q1. for文の役割を説明できますか？
- A. 処理の流れを含めて説明できる（3点）
- B. 繰り返し処理ということは分かる（2点）
- C. よく分からない（1点）

### Q2. if文とfor文を組み合わせて使えますか？
- A. 問題に応じて使い分けられる（3点）
- B. なんとなく書ける（2点）
- C. 書けない（1点）

### Q3. fizzbuzzの処理を自力で書けそうですが？
- A. 自力で書ける（3点）
- B. 調べたりAIに聞けば書ける（2点）
- C. 書けない（1点）

## ③ データベース

### Q1. N+1問題とは何か知っていますか？
- A. 説明できる（3点）
- B. 聞いたことはある（2点）
- C. 知らない（1点）

### Q2. JOINの役割は？
- A. どんな時に使うか説明できる（3点）
- B. 名前は知っている（2点）
- C. 分からない（1点）

## ④ Web基礎

### Q1. GETとPOSTの違いは？
- A. 用途の違いまで説明できる（3点）
- B. なんとなく分かる（2点）
- C. 分からない（1点）

### Q2. 404エラーとは？
- A. 原因と対処が分かる（3点）
- B. 見たことはある（2点）
- C. 分からない（1点）

## スコア判定（例）
- 25〜30点：自走力あり
- 18〜24点：一部理解不足
- 10〜17点：AI依存傾向あり

## 行動提示（例）
- 「AIを使って効率的なコーディングをしていく」
- 「エラーが出たら10分は自力で原因を考える」
- 「for文の問題を1日1問解く」
- 「AIには答えではなくヒントを求める」

## もしも余裕があれば増やすかもしれない質問項目「⑤ 理解度・自走力」

### Q1. 新しい機能を実装するときどうしますか？
- A. 要件を分解して順番に考える（3点）
- B. 参考記事を探して真似する（2点）
- C. AIに丸投げする（1点）

### Q2. 分からない問題にぶつかったときの行動は？
- A. 仮説 → 検証を繰り返す（3点）
- B. とりあえず調べる（2点）
- C. すぐ諦めてAIに頼る（1点）

## どうやって診断機能を作るか
ポイント加点性とする。3問程度の択一の質問として、そこに質問を設けてあらかじめ質問と点数を紐づけておく。また「アルゴリズム」「DB」「Web基礎」「AI活用習慣」のようなカテゴリに質問を分類して、各分野毎に結果を出せるようにする。

例：質問と選択のjson
```js
const questions = [
  {
    question: "AI活用習慣：エラーが出た時、まず何をしますか？",
    choices: [
      { text: "A.エラーを確認して原因について仮説を立てながらググる", score: 3 },
      { text: "B.自分の力だけでコードを一つ一つ見ながら対処していく", score: 2 },
      { text: "C.AIにコードを全貼りする", score: 1 }
    ]
  },
  {
    question: "アルゴリズム：分岐選択繰り返しについて?",
    choices: [
      { text: "if文for文を併用して書ける", score: 3 },
      { text: "if文 が書ける", score: 2 },
      { text: "if文 for文について何も知らない", score: 1 }
    ]
  }
];
```
