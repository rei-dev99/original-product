# 理解負債チェッカー
サブタイトル：〜わかるコード診断〜

## 一言サービスコンセプト（サービスのキャッチコピーを一言で）
AI時代のプログラミング学習診断・改善アプリ。

## 誰のどんな課題を解決するのか？
ターゲット：バックエンド（Ruby on Rails や Laravel 等）を学習中の初学者

- エラーが出ると何も考えずにAIを使ってしまう人
- コードや課題を見ても内容を理解できず、焦りからAIを使ってしまう人
- 理解やスキルの積み上がりを感じられないまま進んでいる人
- ポートフォリオを作ろうとしたら手が止まってしまう人

## なぜそれを解決したいのか？
AIに依存したまま学習を進めると、自力で問題を解決する力が身につかず、AIでも解決できない問題に直面したときに行き詰まってしまう。
また、理解できていない状態で先に進んでも基礎が抜けているため、 応用が効かず、学習内容が定着しない。
その結果、エンジニアとして成長が実感できず、「向いていないのではないか」という不安に繋がる。

## どうやって解決するのか？
- 診断：10〜20問の技術的質問に回答(1つの分野あたり5問程度)
- 結果：AI依存度スコアと理解度の可視化
- アドバイス：診断結果に基づく学習方針の提示

診断を通した結果より自分の理解度を確認する。
適切なアドバイスをもとに学習方法の改善

## 機能要件

### MVP
- ログイン(ゲストログイン)
- 診断履歴の保存
- 診断：10〜20問の技術的質問に回答(1つの分野あたり5問程度)
- 結果：AI依存度スコアと理解度の可視化
- アドバイス：診断結果に基づく全体の傾向
- 行動（分野ごとに提示）

### v2以降(余裕があれば)
- 行動計画の提示と記録
- 学習チェック機能（達成/未達）

## 非機能要件
- ログイン機能
- SPA構成で直感的な操作を提供
- PCメイン（スマホは簡易対応）
- 主要技術：Rails API + Next.js
- 開発環境：Docker
- データベース：Postgresql or MySQL
- デプロイ先：
  - バックエンド：render or heroku or AWS
  - フロント：vercel
- CI/CD（GitHub Actions）
- テストコード
  - Rails API：RSpec
  - Next.js：Jest
- 矛盾のないDB設計を意識
- githubのmainブランチへプッシュされたら自動デプロイ
